# send-message
Simple message using PHP and MYsql

Steps

1) Install WAMP/ LAMP/ MAMP server accordingly to the OS
2) Run the server.
3) Create database using PHPMYADMIN in the local
4) Import tables from the MYSQL( Folder).
5) Create Database connection.
6) Run the code
